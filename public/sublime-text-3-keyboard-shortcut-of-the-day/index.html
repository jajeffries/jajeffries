Sublime Text 3 Keyboard Shortcut of the Day

17.02.2014 13:22 Uncategorized

Alt-Shift-N where N is a number will split your editor into that number of panes. You can then move between them using Alt-N. Numbers 1 to 4 gives you vertical split panes.

  **Alt-Shift-5 **gives you a vertical and horizontal split   **Alt-Shift-8 **gives a horizontal split
